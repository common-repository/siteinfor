== Description ==
**关于SI实时分析*

SI实时分析是一个网站插件，可以实时的显示您的网站在线访客并展示在地图上，您需要做的仅仅是把JS代码嵌入您的网站中。

一目了然网站访客的国家，区域分布。
一目了然访客的来路、访问页面、访问时间。
一目了然访客使用的语言、系统、浏览器。
更多功能开发中（欢迎建议）

JS调用，不影响网站的打开速度 
多种展示样式，全部色彩搭配，美观实用，方便各种主题网站使用。
更多细节可登陆<a  target="_blank" href="http://www.siteinfor.com/tongji" >SI实时分析</a>查看

**插件特性**

* 显示你的网站的实时在线情况
* 在Google Map中显示访客地理位置
* 显示访客来源（搜索引擎或其他）
* 访客浏览器信息
* 访客操作系统
* 访客的IP地址
* 访客的国家以及城市
* 访客目前访问的页面
* 访客来路页面

== Installation ==

使用此插件

1. 点击插件
2. 点击添加插件
3. 搜索`SI实时分析`
4. 点击安装
5. 设置并开始使用

或者

1. 将下载好的zip文件解压缩到`/wp-content/plugins/` 目录下
2. 激活SI实时分析插件

== Frequently Asked Questions ==

**如何改变插件显示位置或颜色**

要想改变插件在网页中的显示位置

1. 进入管理后台 `设置->SI实时分析`.
1. 选择你想要显示的位置
1. 点击`更新插件`

要想改变插件的颜色

1. 进入管理后台 `设置->SI实时分析`.
1. 选择你想要显示的颜色
1. 点击`更新插件`
